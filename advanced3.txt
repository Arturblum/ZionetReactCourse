Homework
Create a form with 10–15 fields.
Form Requirements
- Use the main types of form controls:
 text inputs
 checkbox
 radio buttons
 range slider
 select
 textarea
 any other relevant input types
- The form must include:
 password and confirm password fields
 email field
- Implement field validation using one form library of your choice
 onBlur validation is preferred
- Validation errors must be displayed to the user
- The Submit button must be disabled if the form is invalid
- The form must be accessible (a11y)
- Focus must be visually visible on interactive elements
 Focus styles must not break or shift the layout
- On submit:
 simulate an asynchronous request (fake delay)
 log the submitted form data to the console
- The form must cache non-sensitive data in localStorage
 Sensitive data (e.g., passwords) must NOT be stored
